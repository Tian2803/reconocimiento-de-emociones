# Detección de Emociones en Tiempo Real

Este proyecto de Python utiliza técnicas de visión por computadora y aprendizaje automático para detectar emociones faciales en tiempo real. A través de la cámara del dispositivo, el programa analiza la expresión facial de la persona y muestra una imagen indicando la emoción detectada.
